# Miscellaneous Tools

Simple CLI tools for various CTF challenges.

## Tools

### hash.py
Calculate MD5, SHA1, and SHA256 hashes.
```bash
python hash.py file.txt
```

### convert.sh
Convert data between different formats.
```bash
./convert.sh "Hello" hex
./convert.sh "Hello" base64
```

## Common Patterns
- Hash cracking
- Data format conversion
- Custom encoding schemes
- QR codes
- Puzzles and riddles
